# Fermentary Website

Public-facing website for [Fermentary](https://github.com/Ariteq/Fermentary) — a SwiftUI iOS Pizza Dough Calculator.

Hosted via GitHub Pages: **https://ariteq.github.io/fermentary-website/**

Contains:
- App landing page
- Privacy Policy (DE + EN) — required for App Store submission

## Why a separate repo?

The main Fermentary repo is private. GitHub Pages from private repos requires GitHub Pro. This public repo handles only the public-facing parts (privacy policy, landing) and stays free.
